# tenant-provisioning-portal-mui

React + Vite + TypeScript + MUI + Zustand + MSW

## Quick start (pnpm)

1. Install pnpm if you don't have it:
   ```bash
   npm install -g pnpm
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Initialize MSW service worker (creates public/mockServiceWorker.js):
   ```bash
   pnpm run msw:build
   ```

4. Start dev server:
   ```bash
   pnpm run dev
   ```

Open http://localhost:5173
